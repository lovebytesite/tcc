The First Day of Tiny Code Christmas Extra

Tiny Code Christmas - Day 1 - EXTRA⌗

Welcome to Day 1 of Tiny Code Christmas EXTRA! We’re glad that you’re joining us again on this festive journey into the world of sizecoding. TCC Extra is intended for people who completed Tiny Code Christmas last year and want a new challenge!

If you haven’t already, read the overview to find out what Tiny Code Christmas is all about!

If you’re reading this on the 11th, check out the Field-FX Monday Night Bytes twitch stream at 9pm GMT, you can watch 4 sizecoders program TIC-80 along to an awesome soundtrack! It’s a non-competitive jam and the size limits aren’t enforced but it should give you a good idea of the possibilities of what you can achieve!

The Challenge: Playing with Fire!⌗

The first challenge is to make the smallest possible implementation of a ‘realistic’ fire effect as outlined in the video in 256 characters.

The Expert Challenge!⌗

Make is as tiny as possible!

Sharing is Caring!⌗

If you feel like it, why not share what you’ve done with us on the LoveByte Discord, #lovebyte on IRCnet, or share on Twitter and Mastodon using the hashtag #lovebytetcc
