The Seventh Day of Tiny Code Christmas Extra

Tiny Code Christmas - Day 7 - EXTRA⌗

Welcome to Day 7 of Tiny Code Christmas EXTRA! We’re glad that you’re joining us again on this festive journey into the world of sizecoding. TCC Extra is intended for people who completed Tiny Code Christmas last year and want a new challenge!

If you haven’t already, read the overview to find out what Tiny Code Christmas is all about!

The Challenge: Snowflake!⌗

The first challenge is to make a snowflake, the second challenge is to make the smallest recognisable code that outputs a snowflake!

Sharing is Caring!⌗

If you feel like it, why not share what you’ve done with us on the LoveByte Discord, #lovebyte on IRCnet, or share on Twitter and Mastodon using the hashtag #lovebytetcc
